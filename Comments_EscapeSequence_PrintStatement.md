What are Comments?

Comments are lines in code that are not executed and are used to explain the code.
# This is a comment
print("Hello")

Multi-line comment
"""
This is a
multi-line comment
"""
⚡ Shortcut (in Visual Studio Code)
Ctrl + / → toggle comment (very useful)


What are Escape Sequences?
Special characters used inside strings to format text.
Common Escape Sequences
Escape	Meaning
\n	New line
\t	Tab space
\\	Backslash
\"	Double quote
\'	Single quote

Examples:

print("Hello\nWorld")
👉 Output:
Hello
World

print("Hello\tWorld")
👉 Output:
Hello    World

What is Print?
Used to display output on the screen.

🔹 Basic Example
print("Hello World")

🔹 Multiple Values
print("Hello", "Dev")

name = "Dev"
print("Hello", name)
